## ⚡ If Something Goes Wrong

|---------|----------|

## 🚨 Emergency Fixes

**If screen is completely frozen:**
```
1. Say: "Let me reload this screen"
2. Press F5 (refresh)
3. Wait 5 seconds
4. Continue as if nothing happened
```

**If you can't remember the next screen:**
```
1. Look at this card
2. Click the next sidebar item
3. Take a breath
4. Start narrating about that screen
```

**If judges ask a question mid-demo:**
```
1. Pause demo
2. Answer the question
3. Resume from where you left off
4. Say "Let me continue with the next screen..."
```
